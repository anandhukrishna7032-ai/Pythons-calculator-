# 🧮 Python Calculator

A simple command-line calculator built using Python. This project performs basic arithmetic operations and was created to practice Python fundamentals.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- ⚠️ Handles division by zero

## 🛠️ Technologies Used

- Python 3

## 🚀 How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/python-calculator.git
   ```

2. Navigate to the project folder:
   ```bash
   cd python-calculator
   ```

3. Run the program:
   ```bash
   python calculator.py
   ```
